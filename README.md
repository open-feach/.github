# `.github` — Open Feach Robotics

**Organization-wide defaults and community health files.**

📖 English · 한국어는 아래 ↓

</div>

---

This repository holds **organization-wide configuration** for
[Open Feach Robotics](https://github.com/open-feach). The files here are inherited by every
repository in the org that does not provide its own, so they only need to live in one place.

It is **not** a ROS package — it keeps `main` as its default branch. The `rolling` +
distro-branch model described in the guidelines applies to **package repositories**.

### What's here

| Path | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | The org's public profile page at [github.com/open-feach](https://github.com/open-feach) |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Contribution guide — branch model, DCO sign-off, licensing, commits, style, tests |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Contributor Covenant v2.1 |
| [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) | Default pull request template |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | Bug report / feature request forms |
| [`.gitmessage`](.gitmessage) | Commit message template (Conventional Commits + sign-off) |

Korean versions (`*.ko.md`) sit alongside the English files.

<div align="center">

# `.github` — Open Feach Robotics

**조직 전체 공통 설정과 커뮤니티 상태 파일.**

</div>

이 저장소는 [Open Feach Robotics](https://github.com/open-feach)의 **조직 전체 공통
설정**을 담습니다. 여기 있는 파일은 자체 파일이 없는 org의 모든 저장소에 상속되므로, 한
곳에만 두면 됩니다.

이 저장소는 ROS 패키지가 **아니라서** 기본 브랜치를 `main`으로 둡니다. 기여 가이드에
설명된 `rolling` + distro 브랜치 모델은 **패키지 저장소**에 적용됩니다.

### 구성

| 경로 | 용도 |
|------|------|
| [`profile/README.md`](profile/README.md) | [github.com/open-feach](https://github.com/open-feach)에 보이는 org 공개 프로필 |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | 기여 가이드 — 브랜치 모델·DCO 서명·라이선스·커밋·스타일·테스트 |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Contributor Covenant v2.1 |
| [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) | 기본 PR 템플릿 |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | 버그 신고 / 기능 제안 폼 |
| [`.gitmessage`](.gitmessage) | 커밋 메시지 템플릿 (Conventional Commits + 서명) |

한국어판(`*.ko.md`)은 각 영어 파일 옆에 함께 있습니다.
